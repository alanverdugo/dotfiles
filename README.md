# dotfiles
Configuration files for my environment.

 - To install `.vimrc`:
 - [ ] Install Vundle:
```bash
git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim
```

 - [ ] Copy `.vimrc` to your home:
```bash
cp .vimrc ~/.vimrc
```

 - [ ] Open  `~/.vimrc` with vim and run PluginInstall:
```bash
:PluginInstall
```
